# 安装指南

## 快速开始

### 1. 准备文件

确保项目目录包含以下文件：
- `manifest.json`
- `popup.html`
- `popup.js`
- `popup.css`
- `content.js`
- `background.js`

### 2. 图标文件（可选）

图标文件位于 `icons/` 目录：
- `icon16.png`
- `icon48.png`
- `icon128.png`

如果没有图标文件，插件仍然可以正常工作，只是不会显示图标。

### 3. 安装到Chrome

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择本项目的根目录（包含manifest.json的目录）
6. 插件安装完成！

### 4. 使用插件

1. 点击浏览器工具栏中的插件图标
2. 插件会自动获取当前页面的URL
3. 点击"解析"按钮解析URL参数
4. 开始使用各种功能！

## 常见问题

### Q: 插件无法加载？
A: 检查以下事项：
- 确保所有必需文件都存在
- 检查manifest.json格式是否正确
- 查看Chrome扩展程序页面的错误信息

### Q: 图标显示为灰色？
A: 这是正常的，如果没有图标文件，Chrome会显示默认图标。

### Q: Header注入不生效？
A: 
- 确保已点击"应用Headers"按钮
- 刷新页面后Headers才会生效
- 某些网站可能阻止Header修改

### Q: 拖拽功能不工作？
A: 
- 确保使用现代浏览器（Chrome 88+）
- 尝试刷新插件页面
- 检查浏览器控制台是否有错误

## 卸载

1. 打开 `chrome://extensions/`
2. 找到"URL参数解析器"
3. 点击"移除"按钮

## 更新

1. 下载最新版本
2. 在扩展程序页面点击"重新加载"按钮
3. 或者先卸载再重新安装
