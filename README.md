# URL参数解析器 Chrome插件

一个功能强大的Chrome浏览器插件，用于快速解析、编辑和管理URL参数，支持Header注入。

## 功能特性

### ✨ 核心功能

1. **URL参数解析**
   - 自动解析URL中的所有查询参数
   - 支持手动输入URL进行解析
   - 一键刷新当前页面URL

2. **参数管理**
   - 可视化表单展示所有参数
   - 支持编辑参数名和参数值
   - 支持添加新参数
   - 支持删除不需要的参数

3. **快速搜索**
   - 实时搜索参数名和参数值
   - 高亮显示匹配结果

4. **拖拽排序**
   - 支持拖拽参数项进行排序
   - 可以将重要参数移到最上方

5. **参数固定**
   - 点击📍图标固定重要参数到顶部
   - 固定的参数会始终显示在最上方
   - 固定参数有特殊高亮显示

6. **Header注入**
   - 支持添加自定义HTTP Headers
   - 可以编辑和管理Headers
   - 一键应用Headers到当前页面

7. **数据持久化**
   - 自动保存参数和Headers配置
   - 刷新插件后数据不丢失

## 安装方法

### 从源码安装

1. 下载或克隆本项目到本地
2. 打开Chrome浏览器，进入扩展程序管理页面：
   - 方式1: 在地址栏输入 `chrome://extensions/`
   - 方式2: 点击右上角三点菜单 → 更多工具 → 扩展程序
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的根目录
6. 插件安装完成！

### 图标文件

由于图标文件较大，需要自行创建或下载图标文件：
- `icons/icon16.png` (16x16像素)
- `icons/icon48.png` (48x48像素)
- `icons/icon128.png` (128x128像素)

如果没有图标文件，插件仍然可以正常工作，只是不会显示图标。

## 使用方法

### 解析URL参数

1. 点击浏览器工具栏中的插件图标
2. 插件会自动获取当前页面的URL
3. 或者手动输入/粘贴URL到输入框
4. 点击"解析"按钮
5. 所有参数会以表单形式展示

### 编辑参数

- **修改参数**: 直接在输入框中编辑参数名或参数值
- **添加参数**: 点击"+ 添加参数"按钮
- **删除参数**: 点击参数项右侧的"删除"按钮
- **固定参数**: 点击📍图标固定参数到顶部

### 搜索参数

在搜索框中输入关键词，会实时过滤显示匹配的参数。

### 拖拽排序

1. 将鼠标悬停在参数项上
2. 按住鼠标左键拖动参数项
3. 拖到目标位置后释放
4. 参数顺序会自动保存

### 应用参数

1. 编辑完参数后，点击"应用参数"按钮
2. 当前标签页会自动跳转到新的URL（包含修改后的参数）

### 管理Headers

1. 点击"Headers"标签页
2. 点击"+ 添加Header"添加自定义Header
3. 编辑Header名称和值
4. 点击"应用Headers"按钮应用
5. 刷新页面后Headers会生效

## 技术栈

- **Manifest V3**: 使用最新的Chrome扩展API
- **原生JavaScript**: 无依赖，轻量级
- **CSS3**: 现代化UI设计
- **Chrome Storage API**: 数据持久化
- **DeclarativeNetRequest API**: Header注入

## 权限说明

插件需要以下权限：

- `activeTab`: 获取当前活动标签页信息
- `storage`: 保存用户配置数据
- `declarativeNetRequest`: 使用声明式API注入Headers（Manifest V3推荐方案）
- `downloads`: 导出文件到下载文件夹
- `<all_urls>`: 访问所有网站（用于Header注入）

## 注意事项

1. **Header注入**: Header注入功能需要刷新页面后才能生效
2. **数据存储**: 所有数据存储在本地，不会上传到服务器
3. **URL格式**: 请确保输入的URL格式正确，否则解析会失败
4. **固定参数**: 固定的参数在重新解析URL时会保留，不会被覆盖
